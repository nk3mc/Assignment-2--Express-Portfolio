# Express Portfolio Website

## Project Overview
This assignment is **Assignment 2 – Express.js Portfolio Website** 
It is an enhanced version of my original HTML, CSS, and JavaScript portfolio from Assignment 1.  
This version uses **Express.js** with **EJS templates** to make the website more dynamic, organized, and efficient.

---

## Features
- Developed with **Node.js**, **Express.js**, and the **EJS view engine**.  
- Uses reusable partials for the **header** and **footer** to avoid repetition.  
- Responsive design created with **Bootstrap 5** and a custom **CSS gradient theme**.  
- Displays my key projects, including:  
  - **Personal Portfolio Website** – upgraded to use Express and EJS.  
  - **Chemical Movie Poster Project** – demonstrates creative use of HTML and CSS design concepts.  
  - **AWS Academy Labs** – hands-on experience with cloud computing and virtualization.  
- Interactive **Contact Form** with client-side message validation and alerts.  
- Footer that automatically updates the current year and includes social media links.
